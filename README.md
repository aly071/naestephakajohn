# Welcome Our Forever Beginning

## A simple web app for John Ferino and Stephanie's Wedding

## Description

This web app is created to celebrate the wedding of John Ferino and Stephanie. It provides family and friends with all the essential details about the wedding, including event schedules, venue information, RSVP functionality, gallery, and more.

## Technologies Used

- Vite – for fast development and build process
- TypeScript – for type-safe and maintainable code
- React – for building the interactive user interface
- shadcn-ui – for elegant, reusable UI components
- Tailwind CSS – for styling and responsive design
